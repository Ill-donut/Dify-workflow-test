# Overview

The SMTX Backup and Disaster Recovery backup service backs up an entire virtual machine to a storage repository outside the cluster, such as a NAS server, and can restore the virtual machine to the state at the time of backup by using the data files and configuration files stored in the repository.

![](../assets/installation_guide/installation_guide_003.png)

Backup and recovery can be used in the following scenarios:

- **Data protection**: Prevent data corruption and loss caused by human error, virus infection, logical errors, and other factors.

- **Data reuse**: Use backup data for development, testing, troubleshooting, and more, without affecting the production environment.

- **Data security**: Prevent large-scale data loss caused by ransomware attacks.

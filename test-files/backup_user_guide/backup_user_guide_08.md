# View Backup Repository

In the **Backup and Disaster Recovery** interface, click **Backup Repository** to view all current backup repositories. The following backup repository information is displayed.

  | Field | Description | Example |
  | ----| ---- | ----|
  | Name | The name of the backup repository. | """nfs_repo""" |
  | Connection Status | The connection status between the backup repository and the backup service. | """Connected""" |
  | Description | The description of the backup repository. | """description""" |
  | Type | The type of the backup repository. | """NFS 4""" |
  | Backup Service | The name of the backup service associated with the backup repository. | """bk_service""" |
  | Server Address | The server address information of the backup repository. | """xx.xx.xx.xx"""|
  | Total Capacity | The total capacity of the backup repository. | """100 TiB""" |
  | Available Capacity | The available capacity of the backup repository. | """80 TiB"""|
  | Used Capacity | The used capacity of the backup repository. | """20 TiB""" |
  | Capacity Utilization | The ratio of used capacity to total capacity in the backup repository. | """ 1.25%""" |

Click a row in a backup repository list to view the folder path information of the backup repository in addition to the information shown in the list.

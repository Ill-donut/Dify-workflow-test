# Manually Run a Backup Plan

A created backup plan runs automatically by default according to the configured backup cycle, and it can also be run manually.

**Prerequisites**

There is no other manually running task for the backup plan at present.

**Procedure**

1. In the **Backup Plan** list, select the backup plan to be executed, then click **...** on the right side of the list, or on the backup plan details page, select **Manual Run**.

2. In the **Manually Run Backup Plan** dialog box, select the backup type.

    - If there are virtual machines in the backup plan that do not contain recovery points generated by manual runs, only full backup can be selected.
    - If all virtual machines in the backup plan have recovery points generated by manual runs, full backup or incremental backup can be selected.

3. Click **Manual Run** to submit the task for manually running the backup plan and automatically create the task for backing up the virtual machine. After the backup virtual machine task is created, you can stop the backup task in the Task Center.
